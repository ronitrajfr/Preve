### Features

- Nextjs
- Uploadthing
- Tailwind
- Clerk
- Gemini 

### Setting up locally

```bash
npm install
```

Change `.env.example` to `.env` and `.env.local.example` to `.env.local` , then add the PostgreSQL url (you can get one for free from NeonDB) and clerk & uploadthing keys and grab the Gemini api key.

And then run :
```bash
npm run dev
```
